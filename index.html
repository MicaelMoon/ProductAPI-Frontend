Node endpoints = http://localhost:3000:

/product (Post) = Takes in a json object with properties: (string)"Name" and (double)"Price"

/products (Get) = Returns all products in json format (no parameters)

.NET endpoints = https://dotnet-api-database-connection.azurewebsites.net:

/allProducts (Get) = Returns all products in json format (no parameters)

/searchProduct (Get) = Returns results that matches provided string. Parameters: (string)"Name"

/product (Post) = Takes in a json object with properties: (string)"Name", (double)"Price"

/productName (Put) = Updates the name of one product. Parameters: (string) Old name, (string) New name

/productPrice (Put) = Updates the price of one product. Parameters: (string) Name, (double) New price

/product (Delete) = Takes in a json object with properties: (string) Id, (string) Name, (double) price
